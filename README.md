Linkipin

Linkipin is a simple social media style UI project built using HTML and CSS with a little JavaScript. The goal of this project was to design a cleaner and better looking interface inspired by professional networking platforms.

The project includes small interactive features such as like and dislike buttons and follow request actions where users can accept or decline.

Tech Stack

HTML for structure
CSS for styling
JavaScript for basic interactions

Live Demo

https://linkipin.vercel.app/

How to Run

Clone the repository and open index.html in your browser.
